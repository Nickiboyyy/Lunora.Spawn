# Lunora.Spawn

Ein einfaches Spawn- und Lobby-Plugin für Minecraft Server.

## Features
- **Automatischer Spawn:** Spieler spawnen beim Betreten des Servers immer am gesetzten Spawn-Punkt.
- **Befehle:**
    - `/spawn`: Teleportiert dich zum Spawn.
    - `/lobby`: Alias für `/spawn`.
    - `/setspawn`: Setzt den globalen Spawn-Punkt an deiner aktuellen Position.
- **Berechtigungen:**
    - `lunora.setspawn`: Erlaubt das Setzen des Spawn-Punkts.

## Installation
1. Lade die `.jar` Datei herunter.
2. Schiebe sie in deinen `plugins` Ordner.
3. Starte den Server neu.
4. Nutze `/setspawn` im Spiel, um den Spawn-Punkt festzulegen.
